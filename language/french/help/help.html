Installer / désinstaller


Aucune mesure spéciale nécessaire, suivez le processus d'installation standard - Extrayez le dossier / contact dans le répertoire ../modules. Installez le module via Admin -> Module système -> Modules.

Des instructions détaillées sur l’installation des modules sont disponibles dans Manuel d'exploitation XOOPS

Mode d'emploi


Il n'y a rien à faire dans l'administration, sauf la configuration des préférences.
